## 2024-05-24 - Add icon to Language Switcher
**Learning:** The language switcher button in the sidebar lacked an icon, making it less discoverable and consistent with the theme switcher. Additionally, when the sidebar was collapsed, the text "EN" was displayed without an icon, which didn't look like a typical icon button.
**Action:** Always include an icon (like a Globe) alongside text for language selectors, and ensure the icon is the primary visual indicator when the container (like a sidebar) is collapsed.
